What it does

Built for developers
who don't wait.

Stop babysitting one AI window. Ninja Terminals runs four agents in parallel and lets them hand off work to each other automatically.

Parallel Agents

Run 4+ Claude Code instances simultaneously in a browser-based terminal grid. Each pane gets its own scope — research, build, test, publish — all running at once.

Self-Improving

Measures tool effectiveness, evolves playbooks, and rates what works based on real data. The orchestrator gets smarter with every session.

Zero Config

One command to start. No database, no cloud, no accounts. Runs entirely on your machine using your own Claude API key.

How it works

From goal to shipped
in three steps.

No configuration files. No orchestration boilerplate. Just describe what you want to build.

01
Start the server

One command spins up a local Node server and opens Ninja Terminals in your browser. Your existing Claude API key and MCP configs are auto-detected.

npx ninja-terminals
02
Describe your goal

Type your objective into the orchestrator. It breaks your goal into parallel subtasks and assigns each one to a terminal — automatically.

03
Watch 4 agents execute

Four Claude Code agents run simultaneously while you watch the activity feed. Real-time status detection tells you exactly what each agent is doing.
